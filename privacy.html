Privacy Policy

Project Overview

This website is a non-commercial educational project developed as part of a Web Design laboratory assignment. The primary goal of this application is to demonstrate practical skills in using the ASP.NET Core framework and the Bootstrap CSS library.

Educational Status (KPI)

The project was created by a student of the Igor Sikorsky Kyiv Polytechnic Institute (KPI). All features, including user registration and profile management, are implemented solely for academic purposes to showcase the application of modern web development standards.

Data Collection & Usage

As this is a static-logic educational site (CCV status), please note:

  • No Real Data Needed: Users are encouraged not to provide real personal information, passwords, or emails.
  • Storage: Any data entered into forms is used only to demonstrate the site's functionality (such as displaying a user profile table) and is not shared with third parties.
  • Non-Commercial: This site does not process payments, display advertisements, or track users for commercial gain.
