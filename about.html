About the Project

This website is a practical assignment for a Web Design course. The primary objective was to develop a clean, responsive user interface for a calculator application.

The name ProCalculator stands for "Programmer's Calculator", reflecting the dual purpose of the project: it is designed as a universal tool for both everyday users and software developers. The program has a standard layout for quick everyday calculations and a specialized mode adapted for programming tasks, such as working with different number systems or bitwise operations.

The project was built using HTML5 and CSS3, with a heavy focus on the Twitter Bootstrap framework to ensure a flexible grid system and consistent UI elements across all devices. The backend structure and page routing are managed by ASP.NET Core 8, developed within the Visual Studio 2022 environment.
